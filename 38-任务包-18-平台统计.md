# 任务包 18：平台统计

## 1. 任务名称

平台统计与使用情况第一版

## 2. 任务归属

- 所属层级：平台公共层

## 3. 背景说明

平台成长后，必须知道哪些模块被使用、哪些任务最多、哪些页面最关键，否则后续优化会失去依据。

## 4. 任务目标

建立平台第一版统计能力，为后续优化和决策提供基础数据。

## 5. 本次必须完成

- 模块使用次数统计
- 任务数量统计
- 文件数量统计
- 用户活跃情况基础统计
- 统计展示页第一版
- 时间范围筛选

## 6. 本次明确不做

- 不做复杂 BI 系统
- 不做高级漏斗分析
- 不做实时流式分析

## 7. 用户使用流程

1. 管理员进入统计页
2. 查看平台基础使用情况
3. 按时间范围筛选
4. 判断哪些能力最常用

## 8. 不允许破坏的已确认内容

- 统计必须建立在真实数据上
- 不得为了图表而编造无意义指标

## 9. 完成标准

- 统计页可访问
- 核心指标可查看
- 能为后续优化提供基础依据

## 10. Git 要求

提交信息示例：

`[任务18] 平台：完成统计第一版`
