# 模块页面清单与主流程

## 1. 目的

这份文档用于把第一批模块的页面清单和主流程集中列出来，方便另一个 agent 直接理解“每个模块应该做哪些页、这些页怎么连接”。

## 2. 分析工作模块

### 页面清单

- 工作台页
- 新建分析页
- 上传与校验页
- 结果页
- 历史任务页
- 报告与导出页
- 日志页

### 主流程

1. 工作台页
2. 新建分析页
3. 上传与校验页
4. 发起分析
5. 结果页
6. 报告与导出页
7. 历史任务页

## 3. 对象管理模块

### 页面清单

- 工作台页
- 导入页
- 列表页
- 详情页
- 规则配置页
- 批量操作页
- 导出页
- 日志页

### 主流程

1. 工作台页
2. 导入页
3. 列表页
4. 详情页
5. 规则配置页
6. 批量操作页
7. 导出页
8. 日志页

## 4. 学习训练模块

### 页面清单

- 模块首页
- 训练集页
- 练习页
- 结果页
- 错题页
- 收藏页
- 统计页

### 主流程

1. 模块首页
2. 训练集页
3. 练习页
4. 结果页
5. 错题页 / 收藏页
6. 统计页

## 5. 平台与模块关系

平台至少应提供这些统一入口：

- 平台首页
- 模块中心
- 任务中心
- 文件中心
- 日志中心

模块中的任务、文件、日志入口应优先接入平台中心，而不是完全重复建一套。

## 6. 当前建议

另一个 agent 在开发模块时，应先看：

- `41-模块清单总表.md`
- `对应模块的功能说明文档`
- 本文件

没有这三份对照，不建议直接进入模块开发。
