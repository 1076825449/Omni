# 地址标签参考词库

本文件用于沉淀人工确认的地址标签，供管户分配自动标签规则参考。

当前词库来源：`/Users/peter/Desktop/地址标签.xlsx`，工作表 `导出计数_地址`，共 357 个标签。

## 使用原则

- 正式展示的地址标签仍必须达到 50 户及以上。
- 人工词库只负责识别“同一个地点的不同写法”，不直接绕过 50 户门槛。
- 非拉堡镇乡镇优先按乡镇归并，如百朋镇、成团镇、穿山镇。
- 拉堡镇和柳江城区的大型小区、市场、园区、道路，可以按人工确认词归并。

## 当前已纳入代码词库

完整代码词库位于：

`backend/app/data/address_reference_tags.py`

已包含桌面 Excel 中的 357 个标签，并维护了少量别名归并，例如：

- `启迪柳州科技城`、`启迪(柳州)科技城`、`启迪（柳州）科技城` -> `启迪科技城`
- `丰泽盛景` -> `丰泽盛景广场`
- `柳江大道1号`、`柳江大道商贸城` -> `毅德城`

## 后续维护

如果后续更新 Excel 清单，需要同步更新 `backend/app/data/address_reference_tags.py`，并提升 `ADDRESS_TAG_RULE_VERSION` 触发历史数据重算。
