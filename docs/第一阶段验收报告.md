## 4. 已知问题与未完成项

### 4.1 本阶段已知限制

| 问题 | 影响 | 处理方式 |
|------|------|---------|
| 分析任务为模拟（3秒后自动成功） | 实际分析算法未接入 | 样板目的达成，后续替换算法即可 |
| 文件导出为占位按钮 | 无法真正生成文件 | 后续接入真实文件生成服务 |
| 数据库为 SQLite | 并发能力有限 | 第一阶段目标达成，后续可迁 PostgreSQL |
| 未做 Playwright 自动化测试 | 无浏览器级冒烟测试 | 第二阶段应补齐 |
| 通知/设置/搜索为骨架 | 无真实业务数据 | 第二阶段扩展 |
| 对象管理/学习训练两模块为骨架 | 未接入真实功能 | 按同样规范扩展即可 |

### 4.2 第二阶段建议优先级

1. **补齐对象管理模块**（完整列表型样板）
2. **接入真实分析算法**替换模拟

## 5. 快速验收步骤

### 启动平台

**后端：**
```bash
cd Omni/backend
.venv/bin/uvicorn app.main:app --reload --port 3000
```

**前端：**
```bash
cd Omni/frontend
npm run dev
```

**测试账号：** `admin / admin123`

### 验收路径

1. 打开 `http://localhost:5173`
2. 使用 admin / admin123 登录
3. 平台首页 → 查看模块卡片
4. 进入"模块中心" → 查看三个模块状态
5. 进入"分析工作模块" → 新建分析 → 上传文件 → 发起分析 → 等3秒 → 查看结果
6. 进入"任务中心" / "文件中心" / "日志中心" → 验证记录已写入
7. 退出登录 → 验证自动跳转登录页
8. 未登录直接访问 `/` → 验证自动跳转 `/login`
