# Omni 统一平台用户使用手册

## 登录

访问 http://127.0.0.1:5173 ，输入用户名和密码登录。

默认测试账号：`admin / admin123`

## 首页

登录后进入首页，显示欢迎信息和最近活动。

## 模块中心

点击导航「模块中心」，查看平台全部 7 个正式模块入口：

- **分析工作模块** — 税务案头分析，生成风险报告
- **对象管理模块** — 平台统一对象管理
- **信息查询表** — 纳税人基础信息查询
- **风险记录台账** — 按纳税人归集风险记录
- **学习训练模块** — 税务知识练习
- **数据仪表盘** — 平台运营统计总览
- **定时调度模块** — 定时任务配置与执行

## 任务中心

入口：导航栏「任务中心」

功能：
- 查看所有平台任务的列表（来源模块/类型/状态/创建时间）
- 按关键词、状态、模块、类型筛选
- 点击「详情」查看任务完整信息（文件数/日志数/关联对象）
- 从详情页跳转到来源模块或结果页

## 文件中心

入口：导航栏「文件中心」

功能：
- 上传文件（自动归属到当前模块）
- 按关键词、模块、状态筛选
- 预览 TXT/CSV/JSON/XLSX 文件内容
- 图片/PDF 显示元数据
- 下载文件
- 归档文件（归档后状态变为 archived）

## 日志中心

入口：导航栏「日志中心」

功能：
- 查看所有操作日志（登录/任务创建/文件上传/分析执行等）
- 按关键词、操作类型、模块、成功/失败筛选
- 点击「详情」查看完整操作信息（操作人/时间/目标/结果）

## 通知中心

入口：导航栏「通知中心」

功能：
- 查看通知列表（分析完成/任务执行/导入结果等）
- 按类型（info/success/warning/error）和已读状态筛选
- 点击「标为已读」单条标记
- 点击「全部标为已读」
- 有 target_url 时可点击「查看」直接跳转到相关页面

## 搜索

入口：顶部搜索框

功能：跨任务、文件、日志、通知、对象搜索

## 统计

入口：导航栏「统计」

功能：查看平台任务统计（总数/成功/失败/成功率）、文件统计、活跃模块数

## 设置

入口：顶部导航「设置」

功能：
- 查看当前账号信息（用户名/角色/昵称）
- 备份管理（查看备份记录/下载备份/发起新备份）
- 角色管理（admin 专属，查看和编辑角色权限）

## 帮助

入口：顶部导航「帮助」

## 模块使用指引

### 分析工作模块

1. 新建分析任务 → 填写名称和描述
2. 上传税务资料文件（CSV/TXT/JSON/XLSX/图片/PDF）
3. 系统解析文件，识别字段和缺失提醒
4. （可选）手工补录增值税/企业所得税/个人所得税指标
5. 点击「发起分析」执行分析
6. 查看分析结果（风险列表/证据/整改建议/资料缺口）
7. 对风险结果进行复核（待核实/已确认/误报/已整改/已移交）
8. 导出报告（JSON/TXT分析报告/TXT税务事项通知书）

### 对象管理模块

1. 查看列表，支持分类/状态/标签/关键词筛选
2. 新建对象 → 填写基本信息
3. 编辑/归档对象
4. 批量更新分类/状态/负责人
5. 导入 CSV/XLSX（支持字段别名识别）
6. 查看对象详情（关联文件/日志/来源批次/关联风险）

### 信息查询表

1. 导入纳税人信息 CSV/XLSX
2. 查看纳税人列表和企业画像（识别号/名称/行业/风险等级等）
3. 按税收管理员/管理分局/风险等级/行业统计

### 风险记录台账

1. 单户记录：输入纳税人信息，新增风险记录
2. 批量记录：导入 CSV/XLSX
3. 筛选：关键词/税收管理员/登记状态/风险状态/日期范围
4. 查看风险档案详情（纳税人基本信息/风险时间线/关联文件）

### 学习训练模块

1. 选择训练集
2. 开始练习，逐题作答
3. 完成后查看得分/正确率/错题
4. 收藏题目/取消收藏
5. 查看学习统计（总练习次数/正确数/平均分）

### 数据仪表盘

1. 查看平台联动总览（任务/文件/日志统计）
2. 查看最近活动列表
3. 按时间范围筛选（7天/30天/90天）

### 定时调度模块

1. 新建定时任务，填写名称/说明/Cron表达式/任务类型
2. 使用常用模板快速选择 Cron 表达式
3. 手动执行任务
4. 查看执行历史（执行时间/结果/失败原因）
5. 启用/停用/编辑/删除任务
